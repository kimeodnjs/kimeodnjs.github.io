## 2. Verse1 (2chorus)
**Solo**
```commandline
1st chorus에서 lick1: G, 6th root position, Major&minor pentatonic

2nd chorus에서 lick2:
G, 6th&4th root position, dominant7 chord voicing+approach, Major&minor pentatonic
```
<br>

**Point**
```commandline
1. Shell voicing

2. Shell voicing에서 dominant7(9)의 대리코드 -> major 3rd의 m7(b5)코드 (G7 -> Bm7(b5))

3. 6th root pentatonic position에서 바로 왼쪽인 3rd root position의 1,2번줄도 활용해보자
  (6th root position의 4, 5번줄 = 3rd root position의 1, 2번줄)
  
4. 6th root position을 기준으로 major 3rd와 blue note(b5)의 위치를 다양하게 활용하기

```

## 4. Solo (2chorus)
**Solo**
```commandline
1st chorus:
1 ~ 7bar: G, B.B king box(2nd root position)

7bar: B.B king style로 12fret 넘어 1번줄 root음 찍어주기

7 ~ 8bar: G, 6th root position, Major&minor pentatonic

9 ~ 12bar: G, 6th root position, Major&minor pentatonic
           옥타브 넘어가면서 연주, 마지막은 root음 찍어주기

2nd chorus:           
12 ~ 18bar: G, 6th root position, Major&minor pentatonic
            C7으로 넘어가면서 C7의 Guide Tone과 Shell voicing으로 찍어주기

18 ~ 20bar: G, 6th root position, Major&minor pentatonic, G7 Guide Tone

21 ~ 24bar: G, 6th root position, Major&minor pentatonic, 4th root dim7 chord
```
<br>

**Point**
```commandline
1. root음의 6th root position block = 4th음의 5th root position block
   root음의 2nd root position block = 4th음의 6th root position block
   root scale을 주로 사용하더라도 코드 진행에 맞춰서 Guide Tone이나 Voicing을 찍어주는 쪽으로 활용

2. 블루스 진행에서 diminshed 활용 -> 기억이 안남

3. 6음, b7음 활용 -> Major와 구별되는 Mixolydian의 특징적인 음으로서 모던한 느낌 연출 가능
```
